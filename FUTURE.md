# Future Roadmap (Post-MVP)

These items are planned but not included in the current demo:

## User accounts
- Supabase or Auth0 authentication
- Saved profiles across devices
- Order history and membership management

## Real product catalog
- CMS or database for inventory
- Live pricing and availability
- Stylist approval workflow before delivery

## Virtual try-on
- Integration with AR/VR try-on partners
- Body measurement capture

## Trade-in and resale
- Partner API integration (The RealReal, Vestiaire Collective)
- Automated trade-in valuation

## Admin dashboard
- Stylist portal to review and edit capsules
- Client management and consult scheduling
- Analytics on quiz completion and conversion

## Multi-language
- English and Spanish (Miami market)
- Localized copy and currency
